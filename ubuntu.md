### Code ‡∏ó‡∏î‡∏™‡∏≠‡∏ö (HPC System Check)

‡πÇ‡∏Ñ‡πâ‡∏î‡∏ô‡∏µ‡πâ‡∏à‡∏∞‡∏ó‡∏î‡∏™‡∏≠‡∏ö‡∏Ñ‡∏£‡∏ö‡∏ó‡∏±‡πâ‡∏á **OpenMP** (‡πÄ‡∏ä‡πá‡∏Ñ Core), **OpenBLAS** (‡∏Ñ‡∏π‡∏ì Matrix), ‡πÅ‡∏•‡∏∞ **LAPACK** (‡πÅ‡∏Å‡πâ‡∏™‡∏°‡∏Å‡∏≤‡∏£ Linear)

‡∏™‡∏£‡πâ‡∏≤‡∏á‡πÑ‡∏ü‡∏•‡πå‡∏ä‡∏∑‡πà‡∏≠ `ubuntu_test.c`:

```c
#include <stdio.h>
#include <stdlib.h>
#include <math.h>
#include <omp.h>
#include <cblas.h>
#include <lapacke.h> // Header ‡∏°‡∏≤‡∏ï‡∏£‡∏ê‡∏≤‡∏ô‡∏Ç‡∏≠‡∏á Ubuntu (‡∏ñ‡πâ‡∏≤‡πÉ‡∏ä‡πâ‡∏ß‡∏¥‡∏ò‡∏µ‡∏ó‡∏µ‡πà 1)
// ‡∏ñ‡πâ‡∏≤‡πÉ‡∏ä‡πâ‡∏ß‡∏¥‡∏ò‡∏µ‡∏ó‡∏µ‡πà 2 (Compile ‡πÄ‡∏≠‡∏á) ‡∏≠‡∏≤‡∏à‡∏ï‡πâ‡∏≠‡∏á‡∏õ‡∏£‡∏∞‡∏Å‡∏≤‡∏® Prototype ‡πÄ‡∏≠‡∏á ‡∏´‡∏£‡∏∑‡∏≠‡πÉ‡∏ä‡πâ header ‡∏ó‡∏µ‡πà‡∏°‡∏±‡∏ô‡πÅ‡∏ñ‡∏°‡∏°‡∏≤

// ‡∏õ‡∏£‡∏∞‡∏Å‡∏≤‡∏® Prototype ‡∏™‡∏≥‡∏´‡∏£‡∏±‡∏ö dgesv (‡∏Å‡∏£‡∏ì‡∏µ Compile ‡πÄ‡∏≠‡∏á‡πÅ‡∏•‡πâ‡∏ß‡∏´‡∏≤ lapacke.h ‡πÑ‡∏°‡πà‡πÄ‡∏à‡∏≠)
// ‡πÅ‡∏ï‡πà‡∏ñ‡πâ‡∏≤‡∏•‡∏á liblapacke-dev ‡∏à‡∏∞‡∏°‡∏µ header ‡∏ô‡∏µ‡πâ‡πÉ‡∏´‡πâ
void dgesv_prototype_check() {
    // Dummy function
}

int main() {
    printf("======================================\n");
    printf("   Ubuntu HPC Stack Test \n");
    printf("======================================\n");

    // 1. OpenMP Check
    int max_threads = omp_get_max_threads();
    printf("[1] OpenMP Status:\n");
    printf("    -> Detected Max Threads: %d\n", max_threads);
    
    #pragma omp parallel
    {
        #pragma omp single
        printf("    -> Parallel Region: Active! (Running inside thread)\n");
    }

    // 2. OpenBLAS Check (Matrix Mul)
    int N = 2000;
    double *A = (double*)malloc(N*N*sizeof(double));
    double *B = (double*)malloc(N*N*sizeof(double));
    double *C = (double*)malloc(N*N*sizeof(double));
    
    // Init Data parallelly
    #pragma omp parallel for
    for(int i=0; i<N*N; i++) {
        A[i] = 1.0; B[i] = 2.0; C[i] = 0.0;
    }

    printf("\n[2] OpenBLAS Check (DGEMM N=%d):\n", N);
    double start = omp_get_wtime();
    cblas_dgemm(CblasRowMajor, CblasNoTrans, CblasNoTrans,
                N, N, N, 1.0, A, N, B, N, 0.0, C, N);
    double end = omp_get_wtime();
    
    double gflops = (2.0*N*N*N) / (end-start) / 1e9;
    printf("    -> Calculation Time: %.4f s\n", end-start);
    printf("    -> Performance: %.2f GFlops\n", gflops);

    // 3. LAPACK Check (Linear Solve Ax=b)
    printf("\n[3] LAPACK Check (DGESV):\n");
    // ‡∏™‡∏£‡πâ‡∏≤‡∏á‡∏£‡∏∞‡∏ö‡∏ö‡∏™‡∏°‡∏Å‡∏≤‡∏£‡πÄ‡∏•‡πá‡∏Å‡πÜ 3x3 ‡πÄ‡∏û‡∏∑‡πà‡∏≠‡∏ï‡∏£‡∏ß‡∏à‡∏™‡∏≠‡∏ö‡∏Ñ‡∏ß‡∏≤‡∏°‡∏ñ‡∏π‡∏Å‡∏ï‡πâ‡∏≠‡∏á
    lapack_int n = 3, nrhs = 1, lda = 3, ldb = 3, info;
    lapack_int ipiv[3];
    
    // Matrix A (Column-Major for standard LAPACK call, or Row-Major if using LAPACKE)
    // ‡πÉ‡∏ä‡πâ LAPACKE_dgesv (C Interface) ‡∏à‡∏∞‡∏à‡∏±‡∏î‡∏Å‡∏≤‡∏£ Row-Major ‡πÑ‡∏î‡πâ‡∏á‡πà‡∏≤‡∏¢‡∏Å‡∏ß‡πà‡∏≤
    double A_sys[9] = {
        3.0, 1.0, 1.0,
        1.0, 3.0, 1.0,
        1.0, 1.0, 3.0
    };
    double b_sys[3] = {5.0, 5.0, 5.0}; // ‡∏Ñ‡∏≥‡∏ï‡∏≠‡∏ö‡∏Ñ‡∏ß‡∏£‡πÄ‡∏õ‡πá‡∏ô 1, 1, 1

    printf("    -> Solving 3x3 system...\n");
    
    // ‡πÄ‡∏£‡∏µ‡∏¢‡∏Å‡πÉ‡∏ä‡πâ LAPACKE (C Interface)
    // ‡∏ñ‡πâ‡∏≤‡πÉ‡∏ä‡πâ‡∏ß‡∏¥‡∏ò‡∏µ Source Compile ‡∏≠‡∏≤‡∏à‡∏ï‡πâ‡∏≠‡∏á‡πÄ‡∏õ‡∏•‡∏µ‡πà‡∏¢‡∏ô‡πÑ‡∏õ‡πÄ‡∏£‡∏µ‡∏¢‡∏Å dgesv_ ‡πÅ‡∏ö‡∏ö Fortran binding
    #ifdef LAPACK_ROW_MAJOR 
        info = LAPACKE_dgesv(LAPACK_ROW_MAJOR, n, nrhs, A_sys, lda, ipiv, b_sys, ldb);
    #else
        // Fallback ‡∏´‡∏£‡∏∑‡∏≠‡πÄ‡∏£‡∏µ‡∏¢‡∏Å‡πÅ‡∏ö‡∏ö Fortran style (‡∏ï‡πâ‡∏≠‡∏á transpose ‡∏Ç‡πâ‡∏≠‡∏°‡∏π‡∏•‡∏Å‡πà‡∏≠‡∏ô‡∏ñ‡πâ‡∏≤‡πÄ‡∏õ‡πá‡∏ô Row Major)
        // ‡πÄ‡∏û‡∏∑‡πà‡∏≠‡∏Ñ‡∏ß‡∏≤‡∏°‡∏á‡πà‡∏≤‡∏¢‡πÉ‡∏ô‡∏ï‡∏±‡∏ß‡∏≠‡∏¢‡πà‡∏≤‡∏á‡∏ô‡∏µ‡πâ ‡∏ú‡∏°‡∏à‡∏∞‡∏™‡∏°‡∏°‡∏ï‡∏¥‡∏ß‡πà‡∏≤‡πÉ‡∏ä‡πâ LAPACKE ‡∏ó‡∏µ‡πà‡∏°‡∏≤‡∏Å‡∏±‡∏ö apt ‡∏´‡∏£‡∏∑‡∏≠ OpenBLAS
        // **‡∏´‡∏°‡∏≤‡∏¢‡πÄ‡∏´‡∏ï‡∏∏:** ‡∏ñ‡πâ‡∏≤ compile ‡πÄ‡∏≠‡∏á OpenBLAS ‡∏à‡∏∞‡πÑ‡∏°‡πà‡∏°‡∏µ lapacke.h ‡∏°‡∏≤‡πÉ‡∏´‡πâ‡πÇ‡∏î‡∏¢ default 
        // ‡∏ï‡πâ‡∏≠‡∏á‡πÄ‡∏£‡∏µ‡∏¢‡∏Å dgesv_(&n, &nrhs, ...) ‡πÅ‡∏ö‡∏ö‡πÄ‡∏î‡∏¥‡∏°
        printf("    -> (Skipping Syntax Check for simple demo, assumes LAPACKE)\n");
        info = 0; 
    #endif

    // *‡∏´‡∏°‡∏≤‡∏¢‡πÄ‡∏´‡∏ï‡∏∏: ‡πÉ‡∏ô Code ‡∏à‡∏£‡∏¥‡∏á‡∏ï‡πâ‡∏≠‡∏á‡πÄ‡∏•‡∏∑‡∏≠‡∏Å‡∏ß‡∏¥‡∏ò‡∏µ‡πÄ‡∏£‡∏µ‡∏¢‡∏Å‡πÉ‡∏´‡πâ‡∏ï‡∏£‡∏á‡∏Å‡∏±‡∏ö Library ‡∏ó‡∏µ‡πà‡∏•‡∏á*
    // ‡∏ñ‡πâ‡∏≤‡∏•‡∏á‡πÅ‡∏ö‡∏ö apt-get install liblapacke-dev ‡∏à‡∏∞‡πÉ‡∏ä‡πâ LAPACKE_dgesv ‡πÑ‡∏î‡πâ‡πÄ‡∏•‡∏¢
    
    free(A); free(B); free(C);
    return 0;
}
```
